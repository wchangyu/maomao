AI Mode

To adopt the calculation from system built-in specialist, and continue the optimization via self-learning, not require the interference from user, but still able to maximize the energy saving in end.

Automation Mode

User can change setting base on the actual circumstances , the control device will start to control according to the setting changes and built-in optimization algorithm. In the meanwhile, system is able to fulfill the requirements from end-user, also maximum energy saving.

Manual Mode

User able to start/stop equipment via single-click, same as fix pump’s running frequency, the control device will execute those orders accordingly once received.

Full Domain Setup

Only allow the specialist level to log-in, and to set the fundamental parameters, together with the safety-factor for system operation.
